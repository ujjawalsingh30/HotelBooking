 This is a Hotel booking system where users can book hotel rooms. Users can also add new hotel rooms on this website and manage the hotel bookings from the admin dashboard.
 
 
 *In this MERN Stack project we will create the user Login & registration feature using Clerk. Clerk is a user authentication & user management platform. It also provides beautiful and ready to use UI components like Sign-In, Sign-up and profile component for react app.

We will also add the email functionality in this project so that when user book any hotel they will instantly get the booking confirmation on their email.
